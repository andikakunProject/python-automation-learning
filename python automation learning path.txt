Python automation learning roadmap:

Week 1

Day 1	-> Topic 		: Python Best Practices
	   Task/Objective	: Learn naming, imports, modules, exceptions. Write small scripts with try/except.
	   
Day 2	-> Topic 		: Working with Files
	   Task/Objective	: Read/write files (os, pathlib, csv, json). Automate renaming files.
	   
Day 3	-> Topic 		: CLI Automation
	   Task/Objective	: Learn argparse. Make a script you can run like: python tool.py --name "andika"
	   
Day 4	-> Topic 		: Regex
	   Task/Objective	: Extract emails, phone numbers, URLs from text using re.
	   
Day 5	-> Topic 		: Scheduling automation
	   Task/Objective	: Use schedule or cron to run a script hourly.
	   
Day 6	-> Topic 		: Logging
	   Task/Objective	: Replace print debugging with logging.
	   
Day 7	-> Topic 		: Mini Project: File Cleanup Bot
	   Task/Objective	: Organize folders by file type automatically.

Day 10	-> Topic 		: Selenium Automation
	   Task/Objective	: Automate clicking, typing, logging into sites.
	   
Day 11	-> Topic 		: Scrape tables
	   Task/Objective	: Extract data into pandas DataFrame
	   
Day 12	-> Topic 		: Export scraped data
	   Task/Objective	: Save output to CSV, JSON, SQLite.
	   
Day 13	-> Topic 		: Anti-blocking techniques
